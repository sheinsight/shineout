### Input.Number

基本 API 和上表相同，特定的 API 如下：

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| max | number | 无 | 最大值 |
| min | number | 无 | 最小值 |
| step | number | 1 | 改变数字跨度，可为小数 |
| allowNull | boolean | false | 允许空值 |
| hideArrow | boolean | false | 是否展示增减按钮 |
| coin | boolean | false | 以千位分隔符展示 |

### Input.Password

基本 API 和上表相同，特定的 API 如下：

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| point | string | '•' | 密码符号 |
